## How to use 

```bash
  # Clone this repository
  git clone https://github.com/wilsonmjunior/go-restaurant-web
  
  # Access the project folder 
  cd go-restaurant-web
  
  # Install dependencies
  yarn 
  
  # Run server (json-server)
  yarn server
  
  # Run the application
  yarn start
  
```
